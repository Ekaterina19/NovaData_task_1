# NovaData_task_1
Задание в PostgreSQL на триггеры

ЗАДАНИЕ:
Вы — стажер в компании, где отсутствует система отслеживания изменений пользователей.

Необходимо:
1. Логировать все изменения пользователей (name, email, role),
2. Хранить аудит изменений в отдельной таблице users_audit 
3. Раз в день экспортировать вчерашние данные об измениях в CSV в 3:00 через pg_cron в образе Docker по пути /tmp/users_audit_export_, а далее указываете ту дату, за который этот csv был создан.
4. Создать функцию логирования изменений по трем полям.
5. Создать trigger на таблицу users.
